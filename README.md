🎓 Student Performance Analytics Using AI

This project predicts whether a student is at risk of poor academic performance using Machine Learning.

The main goal is to help teachers and colleges identify struggling students early and take action before it’s too late.

👨‍💻 Author

Bhanu Prakash
Email: marribhanuprakash2005@gmail.com

📌 What This Project Does

Takes student academic data

Trains machine learning models

Predicts whether a student is at risk

Explains why the student is predicted as at risk

This helps institutions support students earlier.

🧠 Models Used

We used three popular machine learning models:

Logistic Regression

Random Forest

XGBoost

The system achieved 94% accuracy on the dataset.

📊 How the Project Works (Step-by-Step)

Collect student data

Clean and prepare the data

Select important features

Train machine learning models

Test the models

Use SHAP and LIME to explain predictions
